======================================
	p<N>         : N번째 트랙 재생 (예: p1)
	n / b           : 다음 / 이전
        vol <0 - 30> : 볼륨 설정 (0=무음, 30=최대)
        eq <mode>  : normal|pop|rock|jazz|classic|bass
        mp3 <N>     : /MP3/000N.mp3 재생
        f <F> <N>   : /FF/NNN.mp3 재생 (예: f 2 7 -> /02/007.mp3)
        state            : 상태/볼륨/EQ/파일 수/현재 파일 출력
        start/stop     : 재생관련
        help            : 도움말
======================================
